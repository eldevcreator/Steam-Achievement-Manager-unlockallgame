# Как собрать модифицированный SAM

## Что добавлено

✅ **Кнопка "Unlock All Games"** в главном тулбаре
- Разблокирует ВСЕ достижения для ВСЕХ игр в библиотеке
- Двойное подтверждение для безопасности
- Показывает прогресс обработки

## Требования

- Windows
- Visual Studio 2019 или новее (или MSBuild)
- .NET Framework SDK

## Сборка через Visual Studio

1. Откройте `SAM.sln` в Visual Studio
2. Выберите конфигурацию **Release | x86**
3. Нажмите **Build > Build Solution** (Ctrl+Shift+B)
4. Готовые файлы будут в папках:
   - `SAM.Picker\bin\x86\Release\` - главное приложение (выбор игры)
   - `SAM.Game\bin\x86\Release\` - окно управления достижениями

## Сборка через командную строку

```bash
# Используя MSBuild
msbuild SAM.sln /p:Configuration=Release /p:Platform=x86

# Или через dotnet
dotnet build SAM.sln -c Release
```

## Запуск

1. Запустите Steam
2. Запустите `SAM.Picker.exe`
3. Выберите игру из списка
4. В открывшемся окне нажмите кнопку **"Unlock All Games"** в тулбаре
5. Подтвердите действие (2 раза)
6. Дождитесь завершения обработки

## Изменения в коде

### SAM.Game/Manager.Designer.cs
- Добавлена кнопка `_UnlockAllGamesButton` в `_MainToolStrip`
- Добавлен разделитель `_ToolStripSeparator3`

### SAM.Game/Manager.cs
- Добавлен метод `OnUnlockAllGames()` который:
  - Получает список всех установленных игр
  - Для каждой игры запрашивает статистику
  - Разблокирует все достижения (пробует ACH_00 до ACH_999)
  - Сохраняет изменения через `StoreStats()`
  - Показывает прогресс и результат

## Примечания

- Обработка всех игр может занять много времени
- Некоторые игры могут не поддерживать достижения
- Изменения необратимы (можно только вручную сбросить через Reset)
- Работает только с установленными играми
